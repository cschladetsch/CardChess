# Assets

They key folders are:

* **App**. The main application code.
* **External**. Third-party tools.
* **Models**. 3D Models. Preferably .blend.
* **Scenes**. 
* **Textures**. Preferably png.
* **Prefabs**. 
* **Sfx**. Sound effects.
* **Music**. Music tracks.
* **Tmp**. Ignored by git.

Each folder has its own _Readme.md_ for more information.
